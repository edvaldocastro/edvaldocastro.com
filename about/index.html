Especialista em banco de dados que trabalha exclusivamente com o SQL Server desde 2006. Trabalhei como PFE (Premier Field Engineer) na Microsoft Brasil e em outras grandes empresas nas áreas de Finanças, Celulose e Educação (Universidade Federal) e atualmente moro e trabalha como Sr. SQL Server Database. Administrador em Dublin, Irlanda.

 

Tenho experiência no planejamento e implantação de soluções de Alta Disponibilidade e Recuperação de Desastres usando o Windows Server Failover Cluster, Always On Availability Groups e estratégias de Backup / Restauração.

Formado com dois cursos de nível superior (Sistemas de Informação | TI para negócios e comércio).

 

Sou o ex-vice-presidente do MTAC (Multi Technical Audience Contributor), uma organização não governamental que provê suporte e projetos de TI para comunidades carentes no Brasil, e também um membro ativo da comunidade SQL Server no Brasil, com apresentações em conferências, organização de eventos, webcasts e também escrevendo em meu blog.
