# Edvaldo Castro Blog Archive

This repository is a static archive of the old WordPress blog for `edvaldocastro.com`.

What is included:

- Separate pages for each published post
- Separate pages for the published static pages
- The original uploads used by the posts
- A searchable archive view

Source:

- WordPress export: `source/edvaldocastro.WordPress.2023-07-13.xml`

Notes:

- The site was converted to static HTML so it can live on GitHub without a traditional WordPress host.
- Some legacy links that pointed to files outside the exported WordPress uploads may still be incomplete if those files were not present in the backup.
