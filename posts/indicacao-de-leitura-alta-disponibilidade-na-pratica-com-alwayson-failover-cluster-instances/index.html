[INDICAÇÃO DE LEITURA] Alta Disponibilidade na Prática com AlwaysOn Failover Cluster Instances

22 Oct 2015 · Comunidade e Eventos, Virtual PASS BR · allwayson fci, AlwaysOn, marcelo fernandes, MTAC, nilton pinheiro, SQL SERVER, sqlserver

Eu estava bastante atrasado para escrever este post, já se foram alguns meses desde que ganhei um concurso relâmpago onde o Nilton Pinheiro deu um Livro "SQL Server 2014 - Alta Disponibilidade na Prática com AlwaysOn Failover Cluster Instances". [caption id="attachment_1797" align="aligncenter" width="720"] Da esquerda para a direita: Marcelo Fernandes, Edvaldo Castro e Nilton Pinheiro[/caption] Após finalmente estar com o livro em mãos (devido a problemas de logística, que eu mesmo causei), eu ainda posterguei um pouco o post sobre o livro, pois eu realmente queria avaliar se o mesmo realmente é bom e se seria recomendável. Pois bem, eu RECOMENDO MUITO a aquisição e leitura deste livro fantástico, pois de forma simples, prática e objetiva o leitor é direcionado através dos passos mais simples para a criação de um ambiente de Alta Disponibilidade com o SQL Server 2014, entendendo cada detalhe daquilo que está configurando. Também no dia 21/10/2015, Fabrício Catae, (Blog | Twitter) um dos maiores profissionais de SQL Server do Brasil, escreveu também um post em seu blog no dia falando sobre o livro, endossando ainda mais esta recomendação A área de HADR (High Availability & Disaster Recovery) é uma das minhas favoritas dentro do mundo de banco de dados, por este motivo eu li o livro com um ponto de vista bem crítico, para avaliar se o mesmo realmente valeria a pena, e confirmei. VALE MUITO

Os autores

Nilton Pinheiro (Site| Twitter) e Marcelo Fernandes (Blog | Twitter) são dois velhos (no sentido literal da palavra) conhecidos da comunidade técnica SQL Server Brasil, ambos detentores do título de MVP (Most Valuable Professional), pelas contribuições que dão voluntariamente à esta mesma comunidade. Ambos os nomes remetem ao assunto Alta Disponibilidade, e ninguém melhor que estes dois para entregarem um conteúdo tão bom quando o presente no livro.

O livro

Infelizmente nem todos podem ter a "sorte" de ganhar um presente como eu tive, porém o livro está acessível nos seguintes sites (além de outros):   Esta é uma recomendação que eu faço particularmente, pois li e aprovei o conteúdo. Espero que todos aqueles que tenham interesse pelo assunto possam adquirir este livro, e tenho certeza que aqueles que o fizerem, não se arrependerão. Se você tem ou já leu o livro, deixe um mini review nos comentários.   Grande abraço,   Edvaldo Castro http://facebook.com/edvaldocastro
