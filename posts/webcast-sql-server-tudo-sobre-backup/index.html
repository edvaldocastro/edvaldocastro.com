WebCast SQL Server – Tudo sobre Backup

*******************************************************************************

Data: quarta-feira, 05 de dezembro de 2012

Horario: 20:30 Duração (prevista): 1 Hora Link para Inscrição: Inscreva-se AQUI Link do grupo SQL Server RS: http://sqlserverrs.com.br/

Tive a honra de ser convidado a fazer uma apresentação para um dos grandes grupos de usuários de SQL Server no Brasil, filiados ao PASS (Professional Association for SQL Server).

Através do membro administrador do grupo, Marcus Bittencourt (Blog | Twitter), recebi o convite com muito prazer e não hesitei em aceita-lo, afinal, grandes nomes do SQL Server no Brasil já passaram por este grupo, com apresentações fantásticas.

O grupo SQL Server RS está fisicamente no Rio Grande do Sul, onde realizam suas reuniões presenciais periodicamente, além dos webcasts com amplo alcance, podendo ter participação irrestrita de todo e qualquer profissional que estiver interessando em trocar experiências e aprender novidades e até mesmos conceitos importantes sobre o SQL Server.

O foco da apresentação será 100% sobre BACKUP no SQL Server, onde tentaremos falar conceitualmente da importância do backup, e outros quesitos que o envolvem além, de muita, muita demonstração prática de tudo o que é possível e imaginável com as opções de BACKUP e RESTORE do SQL Server...

Para aqueles que quiserem ler um pouco mais, escrevi alguns posts (listados abaixo), sobre BACKUP:

  • BACKUP - Você está fazendo corretamente?
  • Política de backup - Implante esta ideia.
  • Full, Diferencial e T-Log - Salve seus dados
  • Restore - A fase crítica do processo
  • Recovery Model - Trate-o com carinho

Além de outros itens, a pré-agenda da apresentação será a seguinte (podendo sofrer alterações):

  • Teorias e Conceitos (Backup, Política de Backup, Política de Restore, Páginas, T-Log, etc)
  • Recovery Model
    • Simple (Circular)
    • Full (Archive)
    • Bulk-Logged
  • Metadados de Backup
    • BackupFile
    • BackupFileGroup
    • BackupMediaFamily
    • BackupMediaSet
    • BackupSet
  • Backup Básico
    • Full
    • Differential
    • Transaction Log
  • Backup Avançado
    • Tail Log (Demos mdf corrompido).
    • Partial
    • Filegroup (Primary Imprescindível, > possibilidade <> administração, Evitar uso)
  • Opções adicionais
    • Copy_Only
    • Compression
    • Standby
    • Stopat
    • Media Set
    • Mirror
    • Verificação de Erros
    • E muito mais

Inicialmente o tempo previsto é de 1 hora de apresentação, mas se a receptividade estiver boa, e todos estiverem de acordo, podemos estender um pouco mais e discutir sobre backup até o entendimento de todos os presentes e interessados...

Desde já, fica o convite para que se inscrevam e divulguem este webcast que será o 8º encontro do Grupo SQL Server RS.
