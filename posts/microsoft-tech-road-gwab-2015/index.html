Microsoft Tech Road GWAB 2015

18 Mar 2015 · Comunidade e Eventos · azure, FoRG, Global Azure Bootcamp, gwab2015, MTAC, sqlserver, Virtual Pass Br, Virtual Pass PT, VirtualPass, VirtualPassBr

Acontece Sábado, dia 25 de Abril de 2015 o Global Azure Bootcamp, um evento global realizado simultaneamente (salvo fusos horários) em mais de 95 cidades em todo mundo. No Brasil, serão 12 cidades sediando o evento, sendo Brasília uma delas. Local: Fapro - (Faculdade Projeção) - Taguatinga - DF Endereço:
  • CNB 14 Lote 7 - Sandú Norte
  • Taguatinga
  • Brasília - DF
  • 72115-145
Data: 25/04/2015 Gostaria de convidar a todos os interessados, sendo de Brasília ou não a prestigiar este grande evento, que tem foco em Cloud Computing (Microsoft Azure). Teremos uma trilha específica de SQL Server, com várias palestras falando do SQL Server integrado com a Nuvem da Microsoft. Para se inscrever basta clicar no link abaixo, e preencher o formulário.
  • Clique para inscrever-se como participante
Se você deseja ser um palestrante, pode submeter sua palestra até o data limite de 27 de Março, pelo link abaixo.
  • Clique para submeter uma palestra
O resultado das palestras selecionadas, será divulgado no dia 03 de Abril pelos mesmos meios de divulgação do evento (site, blogs, facebook, twitter, etc). Qualquer dúvida, fique à vontade para comentar no post. Submeta sua palestra ou venha prestigiar a aprender Cloud Computing neste grande evento totalmente gratuito. Abraços, e até dia 25 de Abril.
