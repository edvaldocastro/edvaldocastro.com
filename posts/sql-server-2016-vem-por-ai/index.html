SQL Server 2016 vem por aí...

05 May 2015 · Comunidade e Eventos, Virtual PASS BR · Comunidade, ignite, MTAC, Pass, sqlfriends, sqlserver, SQLSERVER2016, virtual pass, Virtual Pass Br, VirtualPass

Chicago, 4 de Abril de 2016

Entre os dias 4 e 8 de Abril de 2015 acontece(u) em Chicago, IL - USA o Microsoft Ignite, um grande evento realizado pela Microsoft, onde são anunciados grandes lançamentos e novidades.

Uma das principais novidades, que pode ser de nosso interesse (Profissionais de dados), foi o anúncio da liberação da versão preview pública do SQL Server 2016, previsto para o verão (meados de Junho).

Foram muitas as novidades anunciadas para o produto, todas com um breve descritivo no Data Sheet do mesmo.

Dentre as principais novidades anunciadas, destaco abaixo as que mais me agradaram, e que ao meu ver, vão suprir gaps que ainda existem.

  • AlwaysOn com suporte ao MSDTC (Microsoft Distributed Transaction Coordinator)
    • Pessoalmente, já deixei de implementar grandes projetos de AlwaysOn, por conta desta "deficiência". Transações distribuídas em alguns ambientes são inevitáveis e este era até aqui, uma grande barreira ao uso do SQL Server AlwaysOn Availability Groups.  Bem vindo MSDTC.
  • AlwaysOn com até 3 réplicas síncronas e failover automático
    • Particularmente, não vejo esta como uma das maiores novidades, ao meu ver, a disponibilidade de réplicas com failover automático disponível até o SQL Server 2014 já supre a demanda da maioria dos ambientes, mas disponibilidade nunca é demais, então. Bem vinda 3ª réplica síncrona com failover automático.
  • AlwaysOn com load balancing por round-robing para réplicas secundárias
    • Excepcional, agora além de poder direcionar as aplicações de leitura (relatórios) para as réplicas secundárias, será possível um balanceamento automático de carga para estas réplicas, otimizando ainda mais o processamento destas aplicações de relatório, que geralmente consomem muito recurso.
  • Backup híbridos mais rápidos
    • Backup é um dos pontos de grande interesse pessoal que tenho com relação ao SQL Server, cenários híbridos de Backup e Disaster Recovery foram melhorados (ainda não tenho detalhes), e isto só vem a agregar para a continuidade do negócio....
  • Stretch Database
    • Outra grande novidade com cenários híbridos, esta feature permitirá aumento na escalabilidade, deixando de forma dinâmica dados históricos na nuvem enquanto os dados "quentes" permanecem no on-premisses...
  • e muito mais...

Você pode ler a matéria completa, no blog oficial do SQL Server, onde são lançadas oficialmente as notas relativas à plataforma de dados da Microsoft.

Existe também a possibilidade de ser informado por e-mail quando for liberada a primeira versão preview para download, para isto, basta cadastrar-se neste link:

Get an early look at the next Microsoft data platform

Sem mais, é hora de arregaçar as mangas, e continuar estudando as novidades que vem por aí, e acima de tudo e mais importante de tudo:

"Master the Basics" (desconheço o autor)

Ainda hoje (05/05/2015) estava conversando justamente sobre isso ("Master the basis") com Luciano Moreira [Luti] e como este importante item infelizmente tem sido deixado de lado ultimamente por muitos profissionais do nosso meio...

Aguardemos novidades e fiquemos na expectativa do verão Norte Americano...

Abraços e até a próxima.
