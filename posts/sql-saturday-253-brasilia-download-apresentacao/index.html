SQL Saturday 253 - Brasília - Download apresentação

02 Oct 2013 · Comunidade e Eventos · Brasília, Comunidade, Evento, Pass, SQL Saturday, SQL SERVER, sqlsat253, Virtual Pass Br

Enfim aconteceu o grande evento que todos da comunidade técnica de SQL Server em Brasília estavam esperando.

O SQL Saturday 253 foi um grande sucesso, citado por muitos, como o maior e melhor já realizado em terras tupiniquins, assim sendo só tenho a concordar e confirmar, foi muito bom ver salas cheias, pessoas interessadas, patrocinadores e voluntários trabalhando com dedicação total, para que os presentes pudessem ter (e tivemos) um grandioso evento.

Como já havia informado no post anterior, minha colaboração para o evento, foi uma apresentação sobre "Windows Server Failover Clustering - Fundamentals", onde tentei passar um poucos dos fundamentos de um Windows Cluster aos presentes (sala cheia o/)...

Clique no link abaixo, para fazer o download da apresentação:
