[Agenda de shows] - a.k.a Palestras

10 Nov 2015 · Comunidade e Eventos · Carreira, Comunidade, Friends of RedGate, MTAC, sqlfriends, virtual pass, Virtual Pass Br, VirtualPassBr, VirtualPassPT, Webcast

  A quem interessar possa. Recentemente tive alguns convites recebidos e submissões de palestras aprovadas para alguns eventos que estão por acontecer. Os próximos eventos que participarei como palestrante, que acontecerão ainda em 2015. Deixo o convite a todos para participarem destes eventos e podermos aprender um pouco mais, ou reforçar o conhecimento sobre os temas a serem tratados. Abaixo, um resumo dos eventos, com datas e horários, por ordem de acontecimento.

24 Hours of PASS - Portuguese edition

http://www.sqlpass.org/24hours/2015/portuguese/Registration.aspx Evento online que acontece todos os anos de forma online durante 2 dias (12 horas por dia), com diversas palestras relacionadas à plataforma de dados Microsoft. Pelo terceiro ano consecutivo, terei o prazer de palestrar neste evento mundial que é direcionado principalmente aos países lusófonos. A agenda completa, você pode acessar aqui!
  • Palestra: HandsOn - SQL Server AlwaysOn Availability Groups
  • Data: 18/11/2015
  • Hora: 19:00 (Horário de Brasília)
  • Tipo: Online
  • Link: http://www.sqlpass.org/24hours/2015/portuguese/Registration.aspx

Mundo TI Brasil

https://www.mundotibrasil.com.br/ O mundo TI faz um trabalho excepcional organizando diversas apresentações para a comunidade. Recebi o convite e farei uma apresentação online, conforme dados abaixo.
  • Palestra: Windows Server Failover Cluster - Melhores Práticas
  • Data: 18/11/2015
  • Hora: 22:00
  • Tipo: Online
  • Link: https://www.facebook.com/events/519105204920804/

SQL Saturday 469 - Brasília - DF

http://www.sqlsaturday.com O SQL Saturday é um evento mundial, voltado a profissionais que lidam direta ou indiretamente com produtos da plataforma de dados Microsoft, principalmente o SQL Server.
  • Palestra: HandsOn - SQL Server AlwaysOn Availability Groups
  • Data: 21/11/2015
  • Hora: 09:15
  • Tipo:  Presencial
  • Link: http://www.sqlsaturday.com/469/Sessions/Details.aspx?sid=40894

Em todos ou somente alguns deste eventos, e tantos outros que acontecem na comunidade técnica, aconselho a participação tanto quanto seja possível. É de grande valia para qualquer profissional que quer investir na carreira e fazer um bom networking.   Abraços, Edvaldo Castro SQL Server - Data Platform Consultant MTAC – Microsoft Technical Audience Contributor   ../..// http://www.twitter.com/edvaldocastro02 http://facebook.com/edvaldocastro
