SQL Saturday 253 - Brasília-DF

26 Sep 2013 · Comunidade e Eventos · Comunidade, Evento, Pass, SQL Saturday, sql saturday 253, SQL SERVER, SQL Server DF, sqlsat253, virtual pass, Virtual Pass Br

Após algumas edições de um dos maiores eventos públicos e gratuitos focados 100% em SQL Server em diversas cidades do Brasil, é a vez da nossa querida Brasília receber sua edição do SQL Saturday.

Acontecerá sábado dia 28 de Setembro de 2013, durante todo o dia.

Maiores informações, podem ser obtidas diretamente no site do evento...  (SQL Saturday Brasília)

Já fiz alguns post divulgando outros eventos que aconteceram neste ano e também em anos anteriores, mas não custa repetir.

O SQL Saturday é um evento do PASS (Professional Association for SQL Server), que reúne profissionais locais e também visitantes em um sábado inteiro de palestras de alto nível, muito networking e oportunidade de conhecer e rever grandes amigos, tudo isto totalmente sem custo.

Submeti apenas uma palestra já aos 45 minutos do segundo tempo, e tive a honra de ser aprovado para também dar minha contribuição a este que promete ser um grandioso evento.

Nesta ocasião, falarei um pouco sobre "WSFC - Windows Server Failover Clustering - Fundamentals", conforme abstract a seguir.

O Windows Server Failover Cluster é um pré-requisito para a implementação do AlwaysOn Availability Groups. Nesta sessão, será demonstrado cada componente do WSFC e também discutiremos os principais pontos de atenção durante a configuração do WSFC. Na demonstração, faremos a criação de um Cluster de 3 nós pronto para receber o AlwaysOn Availability Group.
