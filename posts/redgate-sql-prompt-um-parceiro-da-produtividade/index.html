[REDGATE] SQL Prompt, um parceiro da produtividade

08 May 2017 · Database Administration, Tech, Virtual PASS BR

Tenho participado desde 2012 do programa "Friends of RedGate", de uma empresa (A RedGate) baseada no Reino Unido, especializada em criar excelentes ferramentas para administração e desenvolvimento com banco de dados, com uma grande variedade de ferramentas para o SQL Server. Vou fazer a partir desse post, alguns reviews de algumas das muitas ferramentas que eles desenvolveram para facilitar a vida dos profissionais que atuam na administração e desenvolvimento com bancos de dados. Então, me acompanhe aí...

SQL Prompt

Para quem trabalha escrevendo códigos T-SQL, seja para desenvolvimento, criação de rotinas administrativas ou quaisquer outros propósitos, é um investimento baixíssimo, frente ao retorno que o SQL Prompt te propicia. Essa fantástica ferramenta é um superamigo da produtividade.

Principais features:

  • Refatoramento de código
  • Auto formatação de códigos T-SQL
  • Poderoso IntelliSense
  • Code Snippet
  • Abas Coloridas (para diferenciar ambientes PRD, HML, DEV)
  • Histórico de abas do SSMS
Utilizo o SQL Prompt há vários anos, e mesmo não trabalhando a maior parte do tempo com desenvolvimento T-SQL, ele me ajuda muito pois as atividades de administração e suporte também demandam bastante escrita de códigos. Veja abaixo um pequeno vídeo introdutório ao SQL Prompt (Em inglês) E nesse outro vídeo, um desafio (talvez um pouco parcial) entre um profissional utilizando o SQL Prompt e outro sem o SQL Prompt.  

RedGate no Brasil

Apesar de ser uma empresa estrangeira, a RedGate (desenvolvedora do SQL Prompt)  tem uma representação comercial por uma empresa Brasileira, também especializada em SQL Server. A SQLCore, comercializa, provê suporte e treinamentos aos clientes da RedGate no Brasil, que desejarem adquirir o produto por intermedio de uma empresa brasileira.  

Acesse e confira: www.sqlcore.com.br/
