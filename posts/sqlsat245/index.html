SQL Saturday #245 - Rio de Janeiro

20 Jun 2013 · Comunidade e Eventos · Evento, rio de janeiro, SQL Saturday 245, SQL SERVER, sqlsat245, Virtual Pass Br, VirtualPass

A comunidade SQL Server no Brasil mais uma vez vai realizar o SQL Saturday no Rio de Janeiro.

A exemplo do excelente evento que tivemos no ano passado, quando foi realizado o SQL Saturday #127, no dia 14 de abril de 2012, este ano teremos o SQL Saturday #245 que acontecerá no dia 31 de agosto de 2013 na Universidade Veiga de Almeida, no Rio de Janeiro.

O SQL Saturday é um evento do PASS (Professional Association for SQL Server), que conta com um sábado inteiro de palestras de alto nível e excelente conteúdo além de um excelente newtorking com os melhores profissionais de SQL Server do Brasil.

As inscrições para o evento estão abertas e são gratuitas, porém o número de vagas é limitado. Quem se interessar (aconselho fortemente que se interessem), basta acessar a página do evento no site do SQL Saturday e realizar a inscrição...

Abaixo, uma prévia do que já foi submetido como sugestões de palestras e pode estar dentre as escolhidas.
